# Contributing to react-components.com

## Pull Requests

Pull requests should be a branch from `master`.

## Style Guide

### Code

* Use semicolons;
* Commas last,
* 4 spaces for indentation (no tabs)
* Prefer `'` over `"`
* `'use strict';`
* "Attractive"
